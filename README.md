# Chatbot-JKKD
Ini adalah proyek pribadi berupa chatbot lokal dan offline. Chatbot digabung dengan RAG untuk menjawab pertanyaan seputar peminatan jaringan komputer dan komunikasi data (JKKD), Teknik Informatika, Universitas Padjadjaran

Tech Stack:
- Backend: Flask (Python faremwork) dan Vanilla Python
- Frontend: HTML, Vanilla CSS, dan Vanilla JS
- Provider AI: Ollama
- Model AI: Qwen 2.5:1.5b
- Jenis Model: SLM (Small Language Models)
- Framework AI: RAG (Retrieval-Augmented Generation)
- Data: PDF materi kuliah dan file Markdown (.md)
- Database: SQLite Vector
